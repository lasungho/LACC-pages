LACC Privacy Policy

LACC is an AI-powered YouTube creator discovery and research platform. The application uses the official YouTube Data API to retrieve publicly available video and channel metadata such as titles, channel names, view counts, upload dates, durations, and channel links.

This application uses YouTube API Services.

Information LACC Retrieves

LACC retrieves publicly available YouTube metadata only. This may include video titles, channel titles, public channel identifiers, public video identifiers, upload dates, durations, thumbnail URLs, view counts, and links to YouTube pages.

  • LACC does not download YouTube videos or audio.
  • LACC does not collect Google account passwords.
  • LACC does not sell personal information.
  • LACC stores only information needed to operate the local application, such as saved reference channels, folders, notes, review status, and cached public metadata.
  • Users may delete locally stored LACC files from their own device or contact support for privacy-related help.

YouTube API Services

By using LACC, users are also agreeing to be bound by the YouTube Terms of Service.

LACC complies with the YouTube API Services Terms of Service and the YouTube API Services Developer Policies. API-sourced metadata is used for creator discovery, metadata analysis, reference management, and internal research workflows.

Where YouTube API Services policies require refresh, deletion, or limited retention of API-sourced data, LACC is intended to operate in accordance with those requirements during beta development and review.
